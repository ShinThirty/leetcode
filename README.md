# LeetCode

Solutions for LeetCode problems in Python, Scala, Go, and C++.

## Structure

```
problems/
  <problem-number>/
    solution.py
    solution.scala
    solution.go
    solution.cpp
```

Each directory is named by its LeetCode problem number and contains solutions in one or more languages.
